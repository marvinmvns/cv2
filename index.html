2023 - 2024

Robô assistente virtual com IA Generativa inspirado em Marvin, o Androide Paranóide do Mochileiro das Galáxias. Construído com impressão 3D e Raspberry Pi 4, executa modelo Llama 2.0 localmente com processamento de voz em tempo real. Sistema completo de hotword detection, speech-to-text, inferência LLM e text-to-speech.

  • Hardware: Raspberry Pi 4b 4GB, Câmera PS3EYE (microfone), Anel LED programável, Caixa de som
  • Impressão 3D: Corpo e cabeça customizados inspirados no personagem Marvin
  • IA Generativa: Llama 2.0 executando localmente via llama-cpp com otimizações para ARM
  • Pipeline de Voz: Porcupine (hotword detection) → Vosk (speech recognition) → Llama (inferência) → ElevenLabs (TTS)
  • Python: Orquestração completa do sistema, gerenciamento de contexto conversacional
  • Edge Computing: Processamento local de LLM em dispositivo ARM, maximizando privacidade
  • APIs: ElevenLabs para síntese de voz com personalidade do personagem
  • Controle LED: Biblioteca Python para animações sincronizadas com fala

Fluxo de funcionamento:

  1. Detecção do hotword "Marvin" (Porcupine)
  2. Captura e transcrição de voz para texto (Vosk)
  3. Envio para Llama 2.0 com contexto da conversa (llama-cpp-python)
  4. Inferência local do modelo e geração de resposta
  5. Conversão de texto para voz (ElevenLabs)
  6. Reprodução do áudio e animação LED sincronizada
