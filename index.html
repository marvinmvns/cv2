Sistema de Rastreamento Veicular - FastrackGPS

2009 - 2019

Plataforma de rastreamento veicular em tempo real desenvolvida do zero. Sistema pioneiro que integrava dispositivos GPS chineses (TK102) via GPRS, processava dados de geolocalização e exibia em mapas interativos. Atendeu quase 10.000 clientes com modelo de assinatura mensal.

  • Backend: PHP com sockets para recepção de dados GPRS, protocolo proprietário de comunicação com rastreadores GPS
  • Infraestrutura: Cloud computing (Linode) com IP fixo, servidor VPS customizado com alta performance
  • APIs: Google Maps API, Reverse Geocoding para conversão de coordenadas em endereços
  • Frontend: JavaScript com mapas interativos em tempo real
  • IoT: Integração com dispositivos GPS via SMS e GPRS, interpretação de protocolos binários
  • Banco de Dados: Armazenamento de geolocalização, velocidade, altitude, timestamps
  • Modelo de Negócio: SaaS com aluguel mensal de R$ 10,00, gestão de clientes e suporte técnico
